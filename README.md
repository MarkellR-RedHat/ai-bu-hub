# AI BU Hub

Tools, commands, and productivity kits for the Red Hat AI Business Unit.

## Quick Start

Run the onboarding kit to install everything at once:

```bash
git clone https://github.com/MarkellR-RedHat/ai-bu-onboarding-kit.git
cd ai-bu-onboarding-kit
./setup.sh
```

Or browse individual tools at [the hub](https://markellr-redhat.github.io/ai-bu-hub/).
